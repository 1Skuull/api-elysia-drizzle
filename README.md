# Estudo de bun, Elysia e Drizzle

To install dependencies:

```bash
bun install
```

Generate your MySql:

```Generate drizzle
bun run generate
```

To run:

```Migrate drizzle
bun run migrate
```

```bash
bun run dev
```

This project was created using `bun init` in bun v1.0.26. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
